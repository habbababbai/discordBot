# baiBot
Simple Discord bot made with Typescript.
## Commands - executed with prefix '!'
- ping
## Installation
1. Install dependencies
```s
npm install
```
2. Create .env file in root directory and paste you bot token here
```s
TOKEN = paste_your_token_here
```
3. Build project
```s
tsc
```
4. Run
```s
npm run start
```
